# HyGI-Rendering
> **Hybrid Global Illumination Rendering**

- Research-driven project exploring hybrid global illumination techniques,
combining classical rendering pipelines with machine learning approaches for
more efficient rendering with less ressources.

---

## Project Structure

### Core
- **HyGI-engine**  
  Rendering engine implementing hybrid global illumination within the core pipeline.

### Research
- **HyGI-research**  
  Experiments, paper reviews, prototypes, and exploratory projects focused on GI and ML.

### Documentation
- **HyGI-docs**  
  Technical documentation, architecture notes, specifications, and usage guides.

### Machine Learning
- **HyGI-ml**  
  Data preprocessing, CNN training, and trained models for GI optimization.

### Data
- **HyGI-datasets**  
  Curated datasets for training, evaluation, and benchmarking.

### Development
- **HyGI-devlogs**  
  Development logs, design decisions, progress tracking, and encountered issues.
