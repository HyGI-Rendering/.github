# HyGI-Rendering
> **Hybrid Global Illumination Rendering**

- Research-driven project exploring hybrid global illumination techniques,
combining classical rendering pipelines with machine learning approaches for
more efficient rendering with less ressources.

---

## Project Structure

### Core
- **HyGI-engine**  
  Rendering engine implementing hybrid global illumination within the core pipeline. (falcor based)

### Research
- **HyGI-research**  
  Experiments, Result & Statistics, Benchmarking, Paper reviews & sources links / reference.

### Documentation
- **HyGI-docs**  
  Technical documentation, architecture notes, specifications, and usage guides.

### Machine Learning
- **HyGI-ml**  
  Data processing, CNN training and optimisation, and trained models for GI optimization.

### Data
- **HyGI-datasets**  
  Datasets for training, evaluation, and benchmarking.

### Development
- **HyGI-devlogs**  
  Development logs, design decisions, MCOT, progress tracking, and encountered issues.
